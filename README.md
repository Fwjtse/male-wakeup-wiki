## 下载

### Docker

```
docker pull apachecn0/loving-wiki
docker run -tid -p <port>:80 apachecn0/loving-wiki
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install loving-wiki
loving-wiki <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g loving-wiki
loving-wiki <port>
# 访问 http://localhost:{port} 查看文档
```
